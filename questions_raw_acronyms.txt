ICT Acronyms Master List
300 Essential ICT Acronyms with Complete Definitions
Easy, Medium, and Difficult Levels
EASY LEVEL (Acronyms 1-100)
Basic Computing (1-10)
1. CPU
Central Processing Unit - the primary processor executing instructions and performing
arithmetic, logic, control, and input/output operations.
2. RAM
Random Access Memory - volatile memory temporarily storing data and programs currently in
use, providing fast CPU access.
3. ROM
Read-Only Memory - non-volatile memory permanently storing essential boot instructions and
firmware.
4. HDD
Hard Disk Drive - magnetic storage device using rotating platters to store and retrieve data.
5. SSD
Solid-State Drive - flash memory storage without moving parts, providing faster access than
HDDs.
6. GPU
Graphics Processing Unit - specialized processor for rendering graphics and parallel
computations.
7. PSU
Power Supply Unit - converts AC wall power to DC and distributes to computer components.
8. BIOS
Basic Input/Output System - firmware initializing hardware during boot and providing runtime
services.
9. USB
Universal Serial Bus - standard interface connecting peripherals to computers.
10. LAN
Local Area Network - network covering small geographic area (home, office, building).
Networking Basics (11-20)
11. WAN
Wide Area Network - network spanning large geographic areas connecting multiple LANs.
12. IP
Internet Protocol - network layer protocol for addressing and routing packets.
13. TCP
Transmission Control Protocol - reliable, connection-oriented transport protocol.
14. UDP
User Datagram Protocol - connectionless, unreliable transport protocol prioritizing speed.
15. HTTP
HyperText Transfer Protocol - protocol transmitting web pages and data.
16. HTTPS
HTTP Secure - encrypted HTTP using SSL/TLS.
17. DNS
Domain Name System - translates domain names to IP addresses.
18. DHCP
Dynamic Host Configuration Protocol - automatically assigns IP addresses and configuration.
19. FTP
File Transfer Protocol - protocol transferring files between computers.
20. SMTP
Simple Mail Transfer Protocol - protocol sending email between servers.

23. JS

24. URL

25. API

27. XML

28. AJAX

29. SPA

30. PWA
